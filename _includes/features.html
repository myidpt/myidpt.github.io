Features

  • Authentication supports Auth0 and Gitkit, etc.
  • API key validation
  • JSON to gRPC transcoding
  • API-level monitoring, tracing and logging
  • More features coming in the near future: quota, billing, ACL, etc.
  • API service can be implemented in any coding language using any IDLs
  • Supports the deployment on any cloud or on-premise environment
  • Low latency and high throughput
